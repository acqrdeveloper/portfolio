## Automatic tag

```bash

git tag --sort=-v:refname # Listar descendiente

```

## Manually tag

```bash
git tag -a 1.0.X -m "version 1.0.X" # Comentar version tag
git push -f git@github.com:alexchristianqr/portfolio.git master:gh-pages # Link repositorio github
```
